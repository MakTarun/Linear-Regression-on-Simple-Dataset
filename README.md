# Linear-Regression-on-Simple-Dataset


## Description
This project builds a **Linear Regression model** to predict **sales based on advertising spending** using Python and Machine Learning techniques. The project includes data loading, visualization, model training, prediction, and performance evaluation.

## Objective
Predict sales values using advertisement spending data with the help of Linear Regression.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Dataset
The dataset contains advertising expenses (TV, Radio, Newspaper) and corresponding Sales values.

Example columns:
- TV
- Radio
- Newspaper
- Sales

## Steps Performed
1. Import required libraries  
2. Load and clean dataset  
3. Explore data using graphs  
4. Split data into training and testing sets  
5. Train Linear Regression model  
6. Predict sales values  
7. Evaluate model accuracy  

## Skills Learned
- Regression Analysis
- Data Visualization
- Machine Learning Basics
- Model Training & Prediction
- Using Scikit-learn

## Output
The model predicts sales based on advertisement spending and shows the relationship using graphs.
